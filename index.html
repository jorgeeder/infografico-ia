📌 Tipos de Conhecimento
📄

Declarativo

Saber o quê.

Fatos, conceitos, axiomas.

Ex.: Tabela SQL: "cliente_id = 101" · Base de fatos em sistemas especialistas.

⚙️

Procedural

Saber como.

Sequência de passos, algoritmos.

Ex.: Algoritmo de ordenação QuickSort · Funções Python / API REST.

💡

Heurístico

Saber atalhar.

Regras práticas, aproximações.

Ex.: Heurística A* (estimativa de distância) · “Pense passo a passo” (prompt).

🔗

Semântico

Saber o significado.

Relações contextuais, rede de conceitos.

Ex.: Grafo de conhecimento: "Maria → conhece → João" · Embeddings vetoriais.
